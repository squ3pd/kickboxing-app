# Telegram Mini App для кикбоксеров

Система регистрации и анализа физических нагрузок в кикбоксинге.

## Возможности

- ✅ Управление спортсменами
- ✅ Регистрация тренировок с упражнениями
- ✅ Расчет УОИ (удельный объем интенсивности)
- ✅ Таблица нормативов для ввода результатов студентов
- ✅ Итоговая таблица с расчетами по формулам
- ✅ Современный мобильный интерфейс


## Установка и настройка

### 1. Создание Telegram бота

1. Откройте [@BotFather](https://t.me/BotFather) в Telegram
2. Отправьте команду `/newbot`
3. Следуйте инструкциям для создания бота
4. Сохраните полученный токен бота

### 2. Настройка Mini App

1. Отправьте команду `/newapp` боту BotFather
2. Выберите созданного бота
3. Укажите название приложения
4. Загрузите иконку (опционально)
5. Укажите URL вашего приложения (после размещения на хостинге)

### 3. Размещение приложения

#### Вариант 1: GitHub Pages

1. Создайте репозиторий на GitHub
2. Загрузите файлы проекта
3. Включите GitHub Pages в настройках репозитория
4. Используйте полученный URL в BotFather

#### Вариант 2: Vercel/Netlify

1. Зарегистрируйтесь на [Vercel](https://vercel.com) или [Netlify](https://netlify.com)
2. Загрузите проект
3. Получите URL и используйте его в BotFather

#### Вариант 3: Локальный хостинг

Для тестирования можно использовать любой статический хостинг или локальный сервер.

### 4. Настройка формул расчета

В файле `app.js` найдите функции расчета результатов и замените их формулами из PDF файла:

- `calculateSpeedScore()` - расчет оценки скорости
- `calculateEnduranceScore()` - расчет оценки выносливости
- `calculateStrengthScore()` - расчет оценки силы
- `calculateJumpScore()` - расчет оценки прыгучести
- `calculateFlexibilityScore()` - расчет оценки гибкости
- `calculateVOI()` - расчет УОИ (удельный объем интенсивности)

## Использование

1. Откройте бота в Telegram
2. Нажмите на кнопку Mini App или отправьте команду `/start`
3. Используйте навигацию внизу экрана для перехода между разделами

### Добавление спортсмена

1. Перейдите в раздел "Спортсмены"
2. Нажмите "Добавить спортсмена"
3. Введите имя

### Регистрация тренировки

1. Перейдите в раздел "Зарегистрировать тренировку"
2. Выберите спортсмена
3. Укажите дату и тип занятия
4. Добавьте упражнения с указанием типа, продолжительности и ЧСС
5. Сохраните тренировку

### Таблица нормативов

1. Перейдите в раздел "Таблица нормативов"
2. Выберите спортсмена
3. Введите результаты нормативов
4. Нажмите "Сохранить нормативы"
5. Просмотрите итоговую таблицу с расчетами
