# Быстрое размещение на Vercel (если GitHub Pages не работает)

## Шаг 1: Зайдите на Vercel
1. Откройте https://vercel.com
2. Нажмите "Sign Up" и войдите через GitHub

## Шаг 2: Импортируйте проект
1. Нажмите "Add New Project"
2. Выберите "Import Git Repository"
3. Найдите ваш репозиторий `squ3pd/kickboxing-app`
4. Нажмите "Import"

## Шаг 3: Настройте проект
1. Framework Preset: оставьте "Other" или "Vite"
2. Root Directory: оставьте `./`
3. Нажмите "Deploy"

## Шаг 4: Получите URL
Через 1-2 минуты получите URL вида:
`https://kickboxing-app-xxx.vercel.app`

Используйте этот URL в BotFather!
